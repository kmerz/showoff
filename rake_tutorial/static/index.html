The all mighty Rake

  • A simple ruby build program with capabilities similar to make.

or 1000 times reinvented wheel

  • Author: Jim Weirich
  • Project page: http://rubyforge.org/projects/rake

Why rake?

  • it comes with a ruby package..
  • ..or is is easily installed with gem..
  • ..and is widely used

Why not make?

  • Which make? (BSD, GNU, Qt...)?
  • The syntax is not easy to start with

When not rake?

  • When performance is an issue
  • On OpenBSD :)

Lets start

  • $EDITOR Rakefile

first task

desc "Print current git commit hash"
task :version do
  sh "git rev-parse HEAD"
end

make it default

task :default => [ :version ]

save it to a file

desc "Check syntax of foo"
task :syntax do
  begin
    sh 'ruby -c -w -W2 foo.rb > foo.syntax'
  rescue
    puts "Error: #{$!}"
    exit 1
  end
end

but wait there is more!!!

we can do better

    RUBY_SCRIPTS = FileList['**/*.rb']
    RUBY_SCRIPTS.each do |src|
      target = src.ext("syntax")
      file target => src do
        begin
          sh "ruby -c -w -W2 #{src} 2> #{target} > /dev/null"
          puts File.read target
        rescue
          puts "Error: #{$!}"
          exit 1
        end
      end
      desc "Check syntax of all ruby files"
      task :syntax => target
    end

what is FileList

RUBY_SCRIPTS = FileList['**/*.rb']
  • collect all ruby files in this directory and all subdirectories

but wait there is more!!!

FileList = FileList['**/*.rb'].
  exclude("vendor/")

manipulate the file name

RUBY_SCRIPTS.each do |src|
  target = src.ext("syntax")
  • for every file we found we create a target file

but wait there is more!!!

"src/foo.rb".pathmap("%{^src,bin}X.rb")
# => "bin/foo.rb"

require 'rake/clean'
